IMPORTANT! This Automatic Instalation, assume a Clean Install of KSP 1.9.1 or KSP 1.10.1 in order to avoid conflicts with other MODS.
--------------------------------------------------------------------------------------------------------------------------------
First  - Copy all these files to "KSP MAIN DIRECTORY" (sample: C:\SteamLibrary\steamapps\common\Kerbal Space Program)

Second - Run the Script: (@[KSP ROOT]\SteamLibrary\steamapps\common\Kerbal Space Program))
			for KSP 1.9.1  Pmborg-RealFalcons-v2-Install-For_KSP_1.9.1-All_Dependencies.bat
			for KSP 1.10.1 Pmborg-RealFalcons-v2-Install-For_KSP_1.10.1-All_Dependencies.bat

The respective script will download ckan and install all Dependencies for Pmborg-RealFalcons-v2
